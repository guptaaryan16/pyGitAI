pygit commands 
===============

`pygitai` CLI has various commands that help you with the git based VCS projects. My idea is to incorporate as many new LLM API as possible 
and help users to use LLMs to do something useful on their projects. This will give users some less friction about using any type of LLM anywhere with fast integrations and privacy. Hope that the project will create enough usecases and ideas for more use of LLMs to increase productivity.
